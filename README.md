
# Android-Recipe-book
Тестовое задание. API 21+.

Сделано:
+ Парсинг JSON;
+ Сохранение в БД (Room);
+ Вывод списка рецептов, переход по рецепту;
+ Панель поиска;
+ Сортировка рецептов по имени.
 
 Не сделано:
 - Показ нескольких картинок рецепта;
 - Диалоговое окно с увеличенной картинкой;
 - Сортировка рецептов по дате;
 - Сохранение картинки на устройство;
 - Список похожих рецептов.

Установка: Собрать проект в Android Studio или установить APK-file.
